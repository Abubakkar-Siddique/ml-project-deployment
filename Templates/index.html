Wheat Seeds Classification Prediction System

Asslam-o-Alaikum !

  • The Wheat Seeds Classification Prediction System involves the prediction of species given measurements of seeds from different varieties of wheat.

  • It is a binary (2-class) classification problem.

  • Measurements of different geometrical properties of wheat seeds are needed as input.

  • Three different varieties of Wheat Seeds: Kama, Rosa and Canadian, we initially predict from this system.

  • The main aim of this project is to predict that, Wheat seed are from which variety?

Predict Seeds Variety

Instructions

Input Attributes Details
Area - Wheat Seed Area

Possible values of Area:
Min value = 10 & Max value = 22

Perimeter - Wheat Seed Perimeter

Possible values of perimeter:
Min value = 12 & Max value = 18

Compactness - Wheat Seed Compactness

Formula for Compactness: (4*pi*Area) / (perimeter^2)
Possible values of Compactness:
Min value = 0.800 & Max value = 1.000

Length of Kernel - Wheat Seed Kernel

Possible values for Length of Kernel:
Min value = 4.500 & Max value = 7.000

Width of Kernel - Wheat Seed Kernel

Possible values for Width of Kernel:
Min value = 2.00 & Max value = 4.500

Asymmetry Coefficient - Wheat Seed Asymmetry

Possible values of Asymmetry Coefficient:
Min value = 0.500 & Max value = 9.000

Length of kernel groove - Wheat Seed Groove

Possible values for Length of kernel groove:
Min value = 4.00 & Max value = 7.00

Steps to Follow
Step 1 : Enter values of Area, Perimeter, Compactness, Length of Kernel, Width of Kernel, Asymmetry Coefficient, Length of Kernel Groove
Step 2 : Click on Predict Button
Step 3 : Our Wheat Seeds Classification Prediction System will predict Variety of Seed Kama or Rose or Canadian ?
